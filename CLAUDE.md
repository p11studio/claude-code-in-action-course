# Claude Code in Action — Anthropic Course

## Project Purpose

This repo is a hands-on workspace for following the [Claude Code in Action](https://anthropic.skilljar.com/claude-code-in-action) course by Anthropic.

## Course Modules

1. Coding Assistant Architecture
2. Tool Use System
3. Context Management
4. Visual Communication
5. Custom Automation
6. MCP Server Integration
7. GitHub Workflows
8. Reasoning Modes

## Structure

Each module lives in its own folder under `modules/`:

```
modules/
  01-architecture/
  02-tool-use/
  03-context-management/
  04-visual-communication/
  05-custom-automation/
  06-mcp-integration/
  07-github-workflows/
  08-reasoning-modes/
```

- Prerequisites: command-line familiarity, basic Git knowledge
